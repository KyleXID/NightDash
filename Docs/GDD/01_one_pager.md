# Layer 1 - One-pager

## 게임 한 줄 정의
지구에서 마계로 끌려온 주인공이 6개 지역을 순차 정복하며 마왕을 쓰러뜨리는 스토리 기반 뱀서라이크.

## 핵심 판타지
- 물량을 버티고 역으로 지배하는 전투 감각
- 생존뿐 아니라 지역 해방과 서사 전진 체감
- 클래스별 전투 감각 차이

## 차별화 포인트
- 스토리 비중 강화: 지역 정복형 구조
- 주인공 1명 + 클래스 선택 구조
- 6지역 순차 해방 + 보너스 맵
- 위험도 체크리스트로 플레이스타일 변형

## 타깃
- 메인: 빌드/반복 최적화 선호 코어 유저
- 서브: 서사 있는 로그라이트 선호 유저

## 세션 목표
- 일반 런: 15~20분
- 짧은 변형 런: 8~10분

## MVP 스코프
- 지역: 1개(불타는 황야)
- 클래스: 3개(전사, 마법사, 점성술사)
- 공통 무기: 6개
- 패시브: 10개
- 보스: 1개
- 진화: 3개
- 체크리스트: 5개

## MVP 성공 지표
- 1지역 보스 도달율 60%+
- 1지역 보스 클리어율 35%+
- 첫 진화 경험율 50%+
- 평균 1회 플레이타임 12~18분
- D1 재플레이율 35%+
