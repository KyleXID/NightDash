# Appendix - Glossary

- 런(Run): 입장부터 결과 화면까지 한 판의 플레이 단위
- RiskScore: 체크리스트 위험도 합계
- RewardMultiplier: 위험도에 의해 증가한 보상 배율
- 일반 진화: 무기+패시브 조건의 1단 진화
- 심연 진화: 추가 조건이 필요한 2단 진화
- 정복의 잔재: 런 외 성장 재화
- StageTier: 지역 난이도 단계 계수
- 빌드(Build): 런에서 형성된 무기/패시브 조합
- 소프트락: 진행 불가능하지만 앱이 종료되지 않는 상태
- 리그레션: 기존에 되던 기능이 깨지는 현상
