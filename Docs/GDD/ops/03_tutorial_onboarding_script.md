# Ops - Tutorial & Onboarding Script

## 1. 목표
첫 3분 안에 이동/회피/성장/진화 개념을 이해시키고 이탈을 줄인다.

## 2. 트리거 기반 스크립트

### T0 (시작 5초)
- 조건: 런 시작
- 메시지: "이동으로 균열을 피하세요."

### T1 (첫 적 처치)
- 조건: enemyKill >= 1
- 메시지: "적 처치로 경험치를 획득합니다."

### T2 (레벨업 최초)
- 조건: levelUp == 1
- 메시지: "카드를 선택해 빌드를 만드세요."

### T3 (HP 40% 이하)
- 조건: hpRatio < 0.4
- 메시지: "무리하지 말고 이동 경로를 확보하세요."

### T4 (엘리트 등장)
- 조건: eliteSpawn == true
- 메시지: "엘리트를 처치하면 상자를 얻습니다."

### T5 (보스 상자 획득)
- 조건: chestBossOpen == true
- 메시지: "조건이 맞으면 무기가 진화합니다."

## 3. 노출 제한
- 각 튜토리얼 문구는 런 당 1회
- 동일 카테고리 10초 이내 중복 노출 금지

## 4. 스킵 정책
- 옵션에서 튜토리얼 OFF 가능
- OFF 시에도 핵심 경고(보스 등장)는 노출
