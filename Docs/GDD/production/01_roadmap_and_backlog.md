# Production - Roadmap & Backlog

## 1. 목표
MVP를 8주 내 제작하고, 플레이테스트 가능한 안정 빌드를 확보한다.

## 2. 마일스톤

### M1 (주 1~2): 코어 전투 루프
- 이동
- 적 스폰
- 기본 AI 추적
- 자동공격 1종
- 피격/사망

완료 기준:
- 10분 생존 가능한 기본 루프
- 크래시 없이 반복 플레이 가능

### M2 (주 3~4): 성장 시스템
- 경험치/레벨업
- 선택지 3카드
- 무기 3, 패시브 6
- 리롤

완료 기준:
- 빌드 분화가 눈에 보일 것
- 선택지 무효 카드 없음

### M3 (주 5~6): 보스/진화/난이도
- 1지역 보스
- 보스 상자
- 진화 2~3
- 체크리스트 5개

완료 기준:
- 보스 처치-리워드-결과 화면 루프 완성
- 위험도/배율 계산 정확성 검증

### M4 (주 7~8): 메타/품질
- 정복의 잔재
- 전사 메타 트리 1개
- 세이브/로드
- 성능 튜닝

완료 기준:
- 15분 런 안정 동작
- 60FPS 목표 충족(최저 30FPS 장시간 붕괴 없음)

## 3. 우선순위 백로그

## P0 (필수)
- 상태머신/런 관리
- 전투 루프
- 성장 루프
- 보스/결과 루프
- 세이브

## P1 (중요)
- 진화 연출
- 난이도 체크리스트 UI
- 데이터 검증 툴

## P2 (후순위)
- 고급 연출
- 추가 클래스 4종
- 보너스 맵
