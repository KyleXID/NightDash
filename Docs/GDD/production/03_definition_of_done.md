# Production - Definition of Done (DoD)

## 1. 공통 DoD
- 구현 완료 + 단위 테스트/수동 테스트 통과
- 관련 문서(spec/production) 업데이트
- 로그/에러 처리 포함
- 성능 영향 체크 완료

## 2. 시스템별 DoD

## 전투 시스템
- 적/플레이어 피격 계산 정확
- 최소 3000회 히트 시뮬레이션에서 예외 0
- 프레임 급락 없는지 확인

## 성장 시스템
- 레벨업 선택지 중복/무효 없음
- 슬롯 상한/만렙 처리 정상

## 진화 시스템
- 조건 충족/미충족 케이스 테스트 각각 20회 이상
- 심연 진화 판당 1회 제한 준수

## 난이도 시스템
- RiskScore 계산 일치
- RewardMultiplier 계산 오차 0
- 체크리스트 on/off 즉시 반영

## 세이브 시스템
- 저장/로드/버전 마이그레이션 케이스 통과
- 실패 시 백업/복구 동작 확인

## 3. 콘텐츠 DoD
- 데이터 ID 중복 없음
- 누락 참조 없음
- 툴팁/텍스트 현행화 완료
- 플레이테스트 5회 이상 기록

## 4. 배포 전 DoD
- 치명 버그 0
- 회귀 테스트 세트 통과
- 변경 이력 기록 완료
