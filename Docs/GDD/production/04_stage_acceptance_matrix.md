# Production - Stage Acceptance Matrix

## 목적
각 스테이지가 "개발 완료"로 판정되기 위한 최소 기준.

| Stage | 기능 완료 | 밸런스 기준 | 성능 기준 | QA 필수 케이스 |
|---|---|---|---|---|
| 01 | 보스/진화/결과 루프 동작 | 보스 클리어율 35%±10 | 60FPS 목표 | 20케이스 |
| 02 | 중독/속박 기믹 동작 | 보스 도달율 40~55 | 55FPS+ | 20케이스 |
| 03 | 10분 단기 루프 완성 | 클리어율 50~65 | 60FPS 목표 | 15케이스 |
| 04 | 패턴/포격 기믹 동작 | 도달율 30~45 | 55FPS+ | 25케이스 |
| 05 | 한기/빙결 기믹 동작 | 도달율 20~35 | 50FPS+ | 25케이스 |
| 06 | 보스 러시/최종보스 완료 | 클리어율 10~20 | 50FPS+ | 30케이스 |

## 공통 실패 조건
- 크래시/소프트락 1건이라도 발생 시 미통과
- 보상 누락/해금 누락 발생 시 미통과
- 진화 조건 오판정 1건이라도 발생 시 미통과
