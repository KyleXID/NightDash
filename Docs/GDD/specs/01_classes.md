# Specs - Classes

## 공통 전제
- 모든 클래스는 공통 무기 풀 사용 가능
- 클래스별 고유 시작 무기 1개
- 클래스별 고유 패시브 1개
- 클래스별 고유 궁극기 1개

## 클래스 목록
### 1) 전사
- 시작 무기: 마계 대검
- 고유 패시브: 근접 처치 시 방어 스택
- 궁극기: 광전사의 해방
- 곡선: 초반 강, 후반 광역 의존

### 2) 마법사
- 시작 무기: 마계 파열구
- 고유 패시브: 확률 무료 강화
- 궁극기: 차원 붕괴
- 곡선: 초반 약, 후반 강

### 3) 점성술사
- 시작 무기: 별의 낙하
- 고유 패시브: 리롤 비용 0
- 궁극기: 천체 정렬
- 곡선: 중간, 고점 높음

### 4) 성전사
- 시작 무기: 신성 파동
- 고유 패시브: 피격 시 폭발
- 궁극기: 심판

### 5) 사제
- 시작 무기: 빛의 고리
- 고유 패시브: 주기 회복
- 궁극기: 구원

### 6) 궁수
- 시작 무기: 연속 사격
- 고유 패시브: 이동 중 공격 증가
- 궁극기: 폭우 사격

### 7) 건슬링어
- 시작 무기: 리볼버
- 고유 패시브: 재장전 버프
- 궁극기: 난사 모드
