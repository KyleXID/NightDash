# Specs - Weapons & Evolutions

## 진화 규칙
- 기본: 무기 MAX + 요구 패시브 + 상자 획득
- 심연: 진화 무기 + 추가 조건
- 심연 진화는 판당 1개 제한

## 매핑
### 전방 압박형
- 흑염참 + 암흑 강화 -> 심연 흑염참
- 지옥 철퇴 + 근력 강화 -> 파멸의 철퇴
- 연속 난도질 + 민첩 강화 -> 광란의 난도질

### 직선 관통형
- 어둠의 화살 + 탄도학 -> 암흑 관통 폭풍
- 마력 창 + 집중력 -> 차원 관통창
- 분열 탄환 + 관통 강화 -> 지옥 산탄 폭풍

### 원형 보호형
- 회전 칼날 + 확장된 영역 -> 심연의 소용돌이
- 암흑 장벽 + 신성 강화 -> 타락한 성역
- 사슬 낫 + 약점 간파 -> 영혼 수확자

### 랜덤 낙하형
- 유성 낙하 + 유성학 -> 멸망의 유성 폭풍
- 암흑 번개 + 마력 과부하 -> 지옥 낙뢰
- 심연의 촉수 + 암흑 강화 -> 대심연 강림

### 특수형
- 영혼 흡수 + 생명 소모 -> 영혼 포식
- 시간 왜곡 + 전장 가속 -> 시간 붕괴
- 피의 서약 + 피의 계약 -> 혈계 각성
