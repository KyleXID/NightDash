# Specs - Difficulty Checklist

## 구조
- 입장 전 체크박스 UI
- 항목별 위험도 부여
- 총 위험도 기반 보상 배율 적용

## 난이도 구간
- 0~3: 쉬움
- 4~6: 보통
- 7~9: 어려움
- 10+: 지옥

## 체크리스트
### 전투 강화형
1. 마계의 각성: 적 체력 +40%
2. 광기의 물결: 적 이동속도 +30%
3. 군세의 폭증: 적 수 증가
4. 엘리트의 군림: 엘리트 2배
5. 군주 동행: 보스 2체

### 생존 제한형
6. 구원의 부재: 회복 아이템 제거
7. 피의 굶주림: 초당 HP 감소
8. 부서진 방패: 방어력 효율 감소
9. 고립된 전장: 부활 없음
10. 마력 봉쇄: 궁극기 쿨 증가

### 메커니즘 변형형
11. 어둠의 축복: 처치 폭발
12. 느린 세계: 전체 속도 감소
13. 격노한 심연: 저체력 적 강화
14. 균열의 틈: 랜덤 지속피해 지대 생성
15. 영혼 과밀: 경험치 획득 감소

## 보상 연동
- 경험치, 골드, 희귀 상자, 심연 진화 확률 증가
- 특정 콘텐츠는 위험도 임계값으로 잠금 해제
