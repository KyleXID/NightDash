# Specs - Meta Progression & MVP Delivery

## 메타 재화
- 이름: 정복의 잔재
- 획득처: 지역 클리어, 위험도, 보너스 맵, 업적

## 위험도별 획득 예시
- 위험도 0: 100
- 위험도 6: 180
- 위험도 10: 300

## 클래스 트리 원칙
- 클래스별 3갈래
- 노드 15~18 + 엔드 노드 3
- 전부 습득 불가
- 수치 누적보다 플레이스타일 변화 우선

## 기술 기준
- 2D 탑다운 고정
- SO 데이터 중심
- 이벤트 버스 기반 시스템 결합
- 오브젝트 풀링 필수

## MVP 개발 순서
1. 이동 + 스폰 + 기본 AI
2. 자동 공격 1종 + 데미지 처리
3. 경험치/레벨업 UI
4. 무기 3 + 패시브 6
5. 보스 + 보스 상자
6. 진화 2~3
7. 체크리스트 5
8. 메타 트리 1클래스
9. 1지역 완성

## MVP 수용 기준
- 60FPS 목표, 30FPS 이하 장시간 하락 금지
- 15분 런 완주 안정성 확보
- 1지역 루프 크래시 0
- 진화 조건 판정 오류 0
- 보상 배율 계산 오류 0
