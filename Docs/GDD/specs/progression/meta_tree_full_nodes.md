# Specs - Meta Tree Full Nodes

## 공통 규칙
- 클래스별 포인트 트리 3갈래
- 총 15~18 노드
- 엔드 노드 3개
- 전부 획득 불가

## 전사 노드(예시 16)
- 광전 라인(6): 근접피해, 킬 공속, 저HP 공격증가, 광전 유지시간, 처치 폭발, 엔드(피의 난무)
- 방벽 라인(5): 방어, 피해감소, 넉백저항, 피격 회복, 엔드(철벽 유지)
- 심연 라인(5): 진화무기 강화, 화염 도트, 엘리트 추가피해, 리스크 효율, 엔드(군주 각성)

## 마법사 노드(16)
- 파괴(6): 광역계수, 폭발반경, 관통폭발, 보스추가피해, 엔드(차원붕괴 강화), 시전속도
- 집중(5): 쿨감, 리롤, 희귀선택지, XP 보너스, 엔드(정밀집중)
- 심연(5): 전장정지 강화, 진화 확률, 상태이상 피해, 엔드(무한 여파), 마나과부하 보정

## 점성술사 노드(15)
- 예지(5): 리롤, 랜덤효과 안정화, 드랍개선, 엔드(운명선택)
- 천체(5): 낙하형 강화, 범위 증가, 치명 보정, 엔드(천체폭풍)
- 심연(5): 확률 고점 증가, 저점 보정, 심연진화 보정, 엔드(성좌 붕괴), 리스크 완화

## 성전사/사제/궁수/건슬링어
- 각 클래스 동일 규칙으로 15노드 구성
- 각 문서 확장 시 노드 명세 파일 분리 가능 (`meta/{class}.md`)

## 노드 비용 규칙
- Tier1: 1pt
- Tier2: 2pt
- Tier3: 3pt
- End: 5pt

## 선행조건 규칙
- 같은 라인의 이전 Tier 1개 이상 필요
- End 노드는 해당 라인 Tier3 2개 이상 필요
