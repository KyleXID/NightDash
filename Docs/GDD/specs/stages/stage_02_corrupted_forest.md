# Specs - Stage 02: 부패한 숲

## 지역 목적
- DoT(지속 피해) 대응 학습
- 원거리 위협 대응 포지셔닝 학습
- 상태이상(중독) 관리 도입

## 플레이타임
- 16~19분

## 환경/기믹
- 독 포자 지대: 진입 시 2초마다 중독 스택 1
- 덩굴 속박: 랜덤 위치 속박 지대(0.7초)
- 시야 방해 안개: 화면 가장자리 시인성 -15%

## 타임라인
- 0~4분: 근접+저강도 원거리 혼합
- 4~8분: 중독 부여 적 본격 등장
- 8~12분: 이동 방해 기믹 활성화
- 12~15분: 엘리트 2종 교차 등장
- 16분: 보스 등장

## 몬스터
- 부패한 추종자(근접)
- 포자 궁수(원거리 독 투사체)
- 덩굴 맹수(돌진+속박)
- 균사 거체(고체력, 사망 독구름)

## 엘리트
- 부패 감시관: 광역 독 파동
- 심연 수목병기: 직선 돌진 + 넉백

## 보스
- 이름: 역병 군주 베르단
- 페이즈:
  - P1: 부채꼴 독칼날
  - P2(60%): 포자 폭우 + 소환수 2기
  - P3(25%): 전장 독안개 확장

## 보상/연동
- 독 저항형 패시브 체감 구간
- 진화 키워드 `유성학`, `신성 강화` 추천 구간
