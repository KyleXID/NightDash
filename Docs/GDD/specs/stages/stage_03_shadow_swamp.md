# Specs - Stage 03: 그림자 늪

## 지역 목적
- 짧은 템포 런(8~10분)으로 리듬 전환
- 은신/기습 적 대응 훈련
- 고밀도 짧은 파밍 구조 제공

## 플레이타임
- 8~10분

## 환경/기믹
- 늪지 감속: 이동속도 -18%
- 그림자 웅덩이: 일정 주기 폭발
- 은신 웨이브: 특정 구간 적 투명화(피격 시 해제)

## 타임라인
- 0~2분: 감속 환경 적응
- 2~5분: 은신형 적 밀도 증가
- 5~7분: 엘리트 2회 연속 등장
- 8분: 보스 등장

## 몬스터
- 늪지 사냥꾼(은신 근접)
- 그림자 투창병(직선 관통)
- 진창 포식자(광역 도약)

## 엘리트
- 무명의 그림자: 은신 상태에서 기습 3연타

## 보스
- 이름: 심연의 사냥왕 모르
- 특징: 짧은 쿨 주기, 빠른 이동, 체력은 낮은 편

## 보상/연동
- 클리어 보상은 일반 지역 대비 0.8배
- 진화/희귀 상자 확률 +20%
