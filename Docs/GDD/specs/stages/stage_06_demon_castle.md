# Specs - Stage 06: 마왕성

## 지역 목적
- 엔드게임 최종 검증
- 보스 러시 + 최종 마왕전
- 전체 시스템 숙련도 테스트

## 플레이타임
- 18~22분

## 구조
- 1구역: 정예 웨이브
- 2구역: 중간 보스 2연전
- 3구역: 최종 보스전

## 환경/기믹
- 봉인 룬: 비활성 실패 시 적 버프
- 혈월 페이즈: 일정 시간 전체 피해 증가
- 마왕성 균열: 랜덤 고위험 장판

## 중간 보스
- 심연 기사 라크
- 혼돈 주술왕 에른

## 최종 보스
- 이름: 마왕 아스모델
- 페이즈:
  - P1: 근접/원거리 혼합
  - P2(70%): 분신 소환
  - P3(40%): 전장 기믹 강화
  - P4(15%): 광폭화

## 엔딩 분기
- 일반 엔딩: 클리어
- 심연 엔딩: RiskScore 10+ + 심연 진화 획득 + 마왕성 클리어
