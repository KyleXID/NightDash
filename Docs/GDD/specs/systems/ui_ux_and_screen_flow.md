# Specs - UI/UX & Screen Flow

## 1. 전체 화면 플로우
1. Boot
2. MainMenu
3. StageSelect
4. ClassSelect
5. DifficultyChecklist
6. Stage(Playing)
7. Pause
8. Result
9. MetaUpgrade
10. MainMenu 복귀

## 2. MainMenu 필수 요소
- 시작
- 메타 강화
- 설정(오디오, 그래픽, 조작)
- 진행도 표시(지역 해방률)

## 3. StageSelect 필수 요소
- 지역 카드(해금 여부)
- 권장 난이도 표시
- 드랍/보상 힌트

## 4. ClassSelect 필수 요소
- 클래스 카드
- 시작 무기/고유 패시브/궁극기 툴팁
- 난이도 추천 태그(초심자/숙련자)

## 5. DifficultyChecklist 화면
- 체크박스 목록
- 위험도 합계
- 보상 배율 실시간 표시
- 잠금 콘텐츠 해금 조건 안내

## 6. HUD (전투 중)
- 좌상단: HP, 실드, 버프/디버프 아이콘
- 상단 중앙: 타이머, 웨이브 상태
- 우상단: 레벨, 경험치 바, 리롤 횟수
- 하단: 무기/패시브 슬롯, 궁극기 쿨타임
- 중앙 하단: 상호작용 알림(상자, 진화)

## 7. 레벨업 선택창
- 카드 3개 기본
- 리롤 버튼
- 선택 제한 시간 없음(MVP)
- 일시정지 상태에서만 표시

## 8. 보스 상자/진화 연출
- 게임 일시정지
- 붉은 균열 VFX
- 진화 결과 카드 표시
- 확인 후 전투 재개

## 9. Result 화면
- 승패
- 생존 시간
- 처치 수
- 획득 재화
- 위험도/보상 배율
- 해금 항목
- 다음 액션(다시하기/메타 강화/메뉴)

## 10. UX 원칙
- 핵심 정보는 1초 내 파악 가능해야 함
- 위험도/보상 수치는 항상 수치로 명확히 노출
- 실패 원인(즉사, 누적 피해, 기믹 피해)을 결과 화면에서 알려줄 것
