# Specs - UI/UX & Screen Flow

## 1. 전체 화면 플로우
1. Boot
2. MainMenu
3. StageSelect
4. ClassSelect
5. DifficultyChecklist
6. Stage(Playing)
7. Pause
8. Result
9. MetaUpgrade
10. MainMenu 복귀

## 2. MainMenu 필수 요소
- 시작
- 메타 강화
- 설정(오디오, 그래픽, 조작)
- 진행도 표시(지역 해방률)

## 3. StageSelect 필수 요소
- 지역 카드(해금 여부)
- 권장 난이도 표시
- 드랍/보상 힌트

## 4. ClassSelect 필수 요소
- 클래스 카드
- 시작 무기/고유 패시브/궁극기 툴팁
- 난이도 추천 태그(초심자/숙련자)

## 5. DifficultyChecklist 화면
- 체크박스 목록
- 위험도 합계
- 보상 배율 실시간 표시
- 잠금 콘텐츠 해금 조건 안내

## 6. HUD (전투 중)
- 좌상단: HP, 실드, 버프/디버프 아이콘
- 상단 중앙: 타이머, 웨이브 상태
- 우상단: 레벨, 경험치 바, 리롤 횟수
- 하단: 무기/패시브 슬롯, 궁극기 쿨타임
- 중앙 하단: 상호작용 알림(상자, 진화)

## 7. 레벨업 선택창
- 카드 3개 기본
- 리롤 버튼
- 선택 제한 시간 없음(MVP)
- 일시정지 상태에서만 표시

## 8. 보스 상자/진화 연출
- 게임 일시정지
- 붉은 균열 VFX
- 진화 결과 카드 표시
- 확인 후 전투 재개

## 9. Result 화면
- 승패
- 생존 시간
- 처치 수
- 획득 재화
- 위험도/보상 배율
- 해금 항목
- 다음 액션(다시하기/메타 강화/메뉴)

## 10. UX 원칙
- 핵심 정보는 1초 내 파악 가능해야 함
- 위험도/보상 수치는 항상 수치로 명확히 노출
- 실패 원인(즉사, 누적 피해, 기믹 피해)을 결과 화면에서 알려줄 것

## 11. 와이어프레임 구성안 (Title -> HUD -> Result)

### 11.1 Title Screen Wireframe
```text
+----------------------------------------------------------------------------------+
| [NightDash Logo]                                                   [v0.1.0]      |
|                                                                                  |
|                     (배경: 성 + 소용돌이 하늘 + 붉은 창문 광원)                  |
|                                                                                  |
|                                  [ CONTINUE ]                                    |
|                                  [ NEW RUN  ]                                    |
|                                  [ SETTINGS ]                                    |
|                                  [ EXIT     ]                                    |
|                                                                                  |
| [Language: KO/EN]                                             [Press Any Key]    |
+----------------------------------------------------------------------------------+
```
- 버튼 상태: `기본(딥 바이올렛)` / `포커스(크림슨 외곽광)` / `비활성(저채도)`
- 배치 원칙: 로고 우상단 고정, CTA는 중앙 세로 스택

### 11.2 In-Game HUD Wireframe
```text
+----------------------------------------------------------------------------------+
| HP: 128/160 [████████░░]   Shield: 20 [██░░]      Wave 05        12:34          |
| Buff: [B1][B2][B3]                                             Lv 12   XP [███░] |
|----------------------------------------------------------------------------------|
|                                                                                  |
|                           (플레이/전투 뷰포트)                                   |
|                                                                                  |
|                                                                                  |
|                                                                                  |
| [Potion:2] [Dash:READY] [Interact:E]                  Mini Goal: Survive 40s    |
| Skill1 [Q 2.1s]  Skill2 [W READY]  Skill3 [R 18.4s]   Gold 3420   Souls 118     |
+----------------------------------------------------------------------------------+
```
- 우선순위: `생존(HP/실드) > 진행(Wave/Time) > 성장(Lv/XP) > 경제(골드/소울)`
- 붉은색은 위험/경고에만 사용, 기본 정보는 보라/회색 톤

### 11.3 Result Screen Wireframe
```text
+----------------------------------------------------------------------------------+
|                                 [ RUN RESULT ]                                   |
|                                YOU DIED / VICTORY                                |
|----------------------------------------------------------------------------------|
| Stage: Burning Wastes 01                    Time Survived: 14m 22s              |
| Kills: 1,284                                Highest Combo: 67                    |
| Gold Earned: 5,420                          Souls Earned: 186                    |
| Risk Tier: III                              Reward Multiplier: x1.65             |
|----------------------------------------------------------------------------------|
| Cause of Death: Elite charge + burn tick (3.2s burst)                            |
| Unlocks: [Relic Slot +1] [Class: Warden]                                         |
|----------------------------------------------------------------------------------|
| [ RETRY ]                     [ META UPGRADE ]                    [ MAIN MENU ]  |
+----------------------------------------------------------------------------------+
```
- 결과 메시지: 상단 1초 내 인지 가능하도록 대문구+명암 대비 강화
- 하단 CTA 3개는 동일 크기, 기본 포커스는 `RETRY`
