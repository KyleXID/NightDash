# Telemetry - KPI Dashboard Spec

## 1. 핵심 대시보드
- 퍼널: run_start -> boss_spawn -> boss_kill -> run_end(win)
- 스테이지별 승률/평균 레벨
- 클래스별 승률/픽률
- 위험도 점수 분포

## 2. 경보 기준
- Stage01 보스 도달율 < 50% 3일 지속
- Stage06 클리어율 < 7% 또는 > 25%
- 특정 클래스 픽률 > 45%
- 특정 무기 진화 점유율 > 40%

## 3. 리뷰 주기
- 데일리: 핵심 퍼널
- 주간: 밸런스/이탈 원인
- 마일스톤 종료: 전체 회귀 비교
