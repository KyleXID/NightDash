# NightDash --- 통합 게임 디자인 문서 (GDD)

------------------------------------------------------------------------

# 1. 프로젝트 개요

## 1.1 기본 정보

-   장르: 다크 판타지 뱀서라이크 (Survivor-like Roguelite)
-   플랫폼: PC (Steam 1차 목표)
-   엔진: Unity 2D (Top-Down)
-   플레이 타임: 메인 지역 15\~20분 / 보너스 지역 8\~10분
-   목표: 1인 개발 기준, 시스템 중심 설계

## 1.2 한 줄 요약

지구에서 마계로 강제 소환된 주인공이 6개의 지역을 정복하며 성장하고,
최종적으로 마계를 지배하는 다크 서바이벌 게임.

------------------------------------------------------------------------

# 2. 핵심 게임 구조

## 2.1 기본 플레이 루프

1.  지역 선택
2.  클래스 선택
3.  난이도 체크리스트 선택
4.  전투 진행
5.  레벨업 → 무기/패시브 선택
6.  보스 처치 → 진화 발생
7.  정복 포인트 획득
8.  메타 트리 강화
9.  다음 지역 도전

## 2.2 핵심 차별화 요소

-   클래스별 고유 트리 강화
-   체크리스트 난이도 시스템
-   2단계 진화 시스템
-   보너스 맵 해금 구조
-   어둡고 진지한 분위기

------------------------------------------------------------------------

# 3. 세계관 설정

## 3.1 배경

차원 붕괴로 인해 주인공이 마계로 강제 이동. 마계는 6개의 주요 지역으로
구성되어 있으며 각 지역에 군주가 존재.

## 3.2 지역 구성

1.  불타는 황야
2.  부패한 숲
3.  그림자 늪 (짧은 지역)
4.  타락한 성채
5.  절망의 설원
6.  마왕성 보너스 지역: 금지된 도서관 등

# 8. 난이도 체크리스트 시스템

## 카테고리

### 전투 강화형

-   적 체력 증가
-   적 이동속도 증가
-   엘리트 증가
-   보스 2마리

### 생존 제한형

-   회복 아이템 제거
-   초당 체력 감소
-   부활 없음
-   궁극기 쿨 증가

### 메커니즘 변형형

-   적 처치 폭발
-   환경 지속 피해
-   경험치 감소

## 보상

-   위험도 점수 합산
-   보상 배율 증가
-   심연 진화 확률 증가
-   보너스 맵 확률 증가
